## Tools

The runtime prompt inserts the actual built-in tool list from the active boot
configuration.

Use the visible command-execution tool for quick one-shot commands. Use the
visible process-management tools only for long-running daemons or background
workers.

Do not silently switch to a different built-in if the intended one is hidden or
blocked by policy.

Use the active command/tool surface to inspect `skills/` and read `SKILL.md`
files.
